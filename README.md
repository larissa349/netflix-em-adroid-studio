# Netflix Android Studio Project

## Overview
This project is a Netflix-like application built for Android using Android Studio. The aim of this project is to demonstrate knowledge of Android development, including UI design, database management, and network communication.

## Features
- User authentication with email and password.
- Browse movies and TV shows.
- Watch trailers and read descriptions.
- Save favorite shows.

## Technologies Used
- **Android SDK** - The base for building Android applications.
- **Kotlin/Java** - The programming language used for development.
- **Retrofit** - For making network requests.
- **Firebase** - For user authentication and data storage.

## Getting Started
### Prerequisites
- Android Studio installed on your machine.
- An emulator or physical device for testing.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/larissa349/netflix-em-adroid-studio.git
   ```
2. Open the project in Android Studio.
3. Sync the project with Gradle files.
4. Run the application on an emulator or a physical device.

## Usage
- Create an account or log in with existing credentials.
- Explore the catalog and enjoy content!
